# SsisFtpSource
SSIS FTP Custom Data Flow Component

Let you add a FTP data source component in Visual Studio that can stream files from an FTP server.

Server  ftp.example.com
Path		/Some/Path/Some_files_*.txt
User    [username]
Pwd     [password]
